Balise <T***>

<table></table>

La balise <table>   </table> permet de créer un tableau.

<thead></thead>

La balise <thead>   </thead> permet de définir les données de l'entête d'un tableau.

<tbody></tbody>

La balise <tbody>   </tbody> est utilisé de la même façon que la balise <body>   </body> a l'interieur on insere le corp du tableau.

<tr></tr>

La balise <tr>   </tr> permet de définir une ligne. Dans cette balise on peut y trouver les balises <th>   </th> et <td>   </td> que l'on verra par la suite.

<th></th>

La balise <th>   </th> est un initulé de colonne cette balise peut être placée n'importe ou dans un tableau. elle est située dans une balise <tr>   </tr>.

<td></td>

La balise <td>   </td> définit une cellule du tableau. elle est situer dans une balise <tr>   </tr>.

<tfoot></tfoot>

La balise <tfoot>   </tfoot> est utilisée de la même façon que la balise <footer>   </footer> il s'agit du pied du tableau.

Ajout de ligne

Une fois le tableau créé il ne possedera pas de ligne il faudra donc les definir dans la balise <table>   </table> en y ajoutant l'attribut border et lui attribuer une valeur.

Les éléments <input> </input>

Attributs de input text

Name et id

Nom de l'objet et l'identificateur. Le nom a deux utilisations. Il sert à accéder à l'élément, mais devient aussi un nom de variable dans les données de formulaire quand elle sont passées par POST ou GET. Ainsi si le nom est "montext" et le contenu de value "contenu", les données de formulaire contiendront &montext=contenu.


Value

Contiendra le texte entré par l'utilisateur. On peut l'assigner avec un texte par défaut.





Ce contenu peut être assigné dynamiquement, pour ajouter un contenu aux données de formulaire, ou lu dynamiquement par un script. Exemple de lecture:



Size

Largeur initiale du champ d'entrée de texte. Dans le cas présent, la taille correspond à un nombre de caractères (pour les autres objets c'est un nombre de pixels). La largeur en pixels correspond à une largeur moyenne pour les polices de caractères.


Maxlength

Nombre maximal de caractères que l'utilisateur peut entrer, ou que l'on peut assigner à value. Il peut être supérieur à size, dans ce cas le contenu défile dans le champ de texte.


Disabled

L'objet est désactivé au chargement de la page, et affiché en gris.



Pour l'activer, un script assigne l'attribut avec la valeur false.



Readonly

Cet attribut interdit de modifier le contenu de l'attribut value. A la différence de disabled, l'objet est affiché normalement bien que l'utilisateur ne puisse rien entrer.On modifie cet état comme pour l'attribut disabled:



Parmi les attributs hérités: title, style, tabindex, class.


Des balises spécialisées selon les types de données

Password

Il est équivalent à l'objet text avec la différence que le texte assigné à l'attribut value est masqué, et remplacé par des astérisques.



Son utilisation et son fonctionnement sont les mêmes.

Textarea

Il fonctionne comme text, mais affiche plusieurs lignes et la valeur est contenue entre la balise ouvrante et fermante. Attributs:

  1. Row

    Nombre de lignes visibles. Remplace l'attribut size.

  2. Cols

    Nombre de caractères par ligne. La largeur en pixels correspond à la largeur moyenne.

les evènements

Onfocus

Réagit quand l'objet obtient le focus, donc quand l'utilisateur clique sur le champ de texte.

Onblur

Le focus est perdu (l'utilisateur clique ailleurs).

Onchange

Quand un caractère est entré ou modifié mais est déclenché quand le champ de texte perd le focus.

Onselect

Une partie du texte est sélectionnée.
On commande aussi la sélection du texte par la méthode select().



Onclick

L'utilisateur clique dans le champs de texte.

Acceskey

Hebergeur WEB


Il existe differente solution pour héberger un site web
Je vais vous montrer un tutoriel pour pouvoir heberger votre site avec Github

1. Les sites d'hébergement gratuit

         Il existe des sites d'hébergement gratuit comme github ou 000webhost.
Je vais vous montrez comment héberger votre site WEB sur guthub.

1.Créer un compte GitHub

         Rendez-vous sur le site https://github.com

2.Créer le repository GitHub en terminant par .github.io exemple "Roucky0798.github.io"

         Créer un nouveau repository pour votre site web : https://github.com (exemple : SitePerso.github.io)

3.Télécharger le logiciel Git Bash à l'adresse suivante : https://git-scm.com

4.Ouvrez le logiciel et éffectuez ces commandes dans la fenêtre de commande

         1. cd documents

         2. Faite un git clone puis coller l'URL du clone de votre repository. Dans mon cas cela donne ça :          "git clone https://github.com/Roucky0798/roucky0798.github.io.git"

         3. cd puis coller le nom de votre repository. Dans mon cas : "cd roucky0798.github.io"

         4. un dossier avec le nom de votre repository se trouvera dans vos dossiers documents. Placez-y               les dossiers du site

         5. git add .

         6. git commit -m "Site"

         7. git config --global user.email "Votre adresse mail"

         8. git config --global user.name "Votre pseudo Github"

         9. git commit -m "Site"

         10. git push origin master

5.Rendez-vous sur votre navigateur et rentrer dans l'URL : pseudo.github.io pour mon cas roucky0798.github.io

3. Les hébergement sur serveur

Il existe d'autre moyen d'herbergement de site web comme les serveurs dédié.


Il existe différentes offres d’entreprises qui vous proposent d’héberger votre site web. Autrement dit, ils vous proposent de louer un espace de stockage sur leur serveur pour votre site web.



Pour choisir un bon serveur il faut prendre en compte de la connection de l'hébergeur, le stockage, la ram et le processeur. Sur l'image si dessus on peut voir que le serveur choisit posséde de bonnes performances
