Balise <T***>

<table></table>

La balise <table>   </table> permet de créer un tableau.

<thead></thead>

La balise <thead>   </thead> permet de définir les données de l'entête d'un tableau.

<tbody></tbody>

La balise <tbody>   </tbody> est utilisé de la même façon que la balise <body>   </body> a l'interieur on insere le corp du tableau.

<tr></tr>

La balise <tr>   </tr> permet de définir une ligne. Dans cette balise on peut y trouver les balises <th>   </th> et <td>   </td> que l'on verra par la suite.

<th></th>

La balise <th>   </th> est un initulé de colonne cette balise peut être placée n'importe ou dans un tableau. elle est située dans une balise <tr>   </tr>.

<td></td>

La balise <td>   </td> définit une cellule du tableau. elle est située dans une balise <tr>   </tr>.

<tfoot></tfoot>

La balise <tfoot>   </tfoot> est utilisée de la même façon que la balise <footer>   </footer> il s'agit du pied du tableau.

Ajout de ligne

Une fois le tableau créé il ne possedera pas de ligne il faudra donc les definir dans la balise <table>   </table> en y ajoutant l'attribut border et lui attribuer une valeur.

Hebergeur WEB


Il existe differente solution pour héberger un site web
Je vais vous montrer un tutoriel pour pouvoir heberger votre site avec Github

1. Les sites d'hébergement gratuit

         Il existe des sites d'hébergement gratuit comme github ou 000webhost.
Je vais vous montrez comment héberger votre site WEB sur guthub.

1.Créer un compte GitHub

         Rendez-vous sur le site https://github.com

2.Créer le repository GitHub

         Créer un nouveau repository pour votre site web : https://github.com (exemple : SitePerso)

3.Télécharger le logiciel Git Bash à l'adresse suivante : https://git-scm.com

4.Ouvrez le logiciel et éffectuez ces commandes dans la fenêtre de commande

         1. cd documents

         2. Faite un git clone puis coller l'URL du clone de votre repository. Dans mon cas cela donne ça :          "git clone https://github.com/Roucky0798/roucky0798.github.io.git"

         3. cd puis coller le nom de votre repository. Dans mon cas : "cd roucky0798.github.io"

         4. un dossier avec le nom de votre repository se trouvera dans vos dossiers documents. Placez-y               les dossiers du site

         5. git add .

         6. git commit -m "Site"

         7. git config --global user.email "Votre adresse mail"

         8. git config --global user.name "Votre pseudo Github"

         9. git commit -m "Site"

         10. git push origin master

5.Rendez-vous sur votre navigateur et rentrer dans l'URL : pseudo.github.io pour mon cas roucky0798.github.io

3. Les hébergement sur serveur

Il existe d'autre moyen d'herbergement de site web comme les serveurs dédié.


Il existe différentes offres d’entreprises qui vous proposent d’héberger votre site web. Autrement dit, ils vous proposent de louer un espace de stockage sur leur serveur pour votre site web.



Pour choisir un bon serveur il faut prendre en compte de la connection de l'hébergeur, le stockage, la ram et le processeur. Sur l'image si dessus on peut voir que le serveur choisit posséde de bonnes performances

Achat d'un nom de domaine

Pour héberger son site sur un serveur dédié il vous faudra un nom de domaine et cela n'est pas gratuit pour quelque euro vous pouvez avoir un nom de domaine.
