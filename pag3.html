Le langage XML

Le langage XML qui signifie Extensible Markup Language est un langage qui ressemble au HTML grâce au balisage.
Le nom des balises explique ce que les balises contienent. Tout comme le HTML il possede une balise racine.


Domaine d'application

Le XML est utilisé le plus souvent pour la configuration de logiciel comme par exemple VirtualBox
